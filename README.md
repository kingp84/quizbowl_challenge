# Quizbowl Challenge

An adaptive educational app with modular gameplay, premium features, and celebratory UI.
Built with Python and Streamlit to empower learners and coaches through joyful, accessible tools.

## Features
- Modular loaders for diverse question formats
- Adaptive UI/UX for gamified learning
- Multiplayer support via Streamlit Cloud

## Getting Started
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run locally: streamlit run app.py
